Experimental Courses

These are experimental courses I've developed to complement the standard curriculum.

Fall 2022

CS 85A: Applied Algorithms I

  • Units: 1 unit, Credit/No Credit
  • Format: Weekly 50-minute meetings
  • Focus: Practical application of CS 46B concepts using Java
  • Audience: Designed for both beginners and advanced programmers

Combined with CS 185A (spring) counts as one CS elective.

CS 185C: Distributed Systems

  • Units: 3 unit CS elective
  • Focus: Challenges and solutions in distributed systems
  • Projects: Four programming projects using Java and gRPC

Explores the fundamental challenges of building reliable distributed systems.

Spring 2023

CS 185A: Applied Algorithms II

  • Units: 1 unit, Credit/No Credit
  • Format: Weekly 50-minute meetings
  • Focus: Applies CS 146 concepts through problem-solving
  • Language: Java-based implementations

Targets students at varying skill levels, building on Applied Algorithms I.
