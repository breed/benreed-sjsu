# Starting C++

a short booklet to help you get started with C++. each chapter is meant to help you understand a topic, but you will still want to reference API descriptions for more specifics.

## chapters

- [Chapter 0: How to Use This Booklet](ch00.pdf) — how to get the most out of this booklet, including tips, function signatures, try-it exercises, and end-of-chapter exercises.
- [Chapter 1: Introduction](ch01.pdf) — your first C++ program, compiling and running, namespaces, `std::cout`, `std::cin`, and command-line arguments.
- [Chapter 2: Variables](ch02.pdf) — basic types, declaring variables, `sizeof`, arrays, `const`, and structures.
- [Chapter 3: Strings](ch03.pdf) — the `std::string` type, length, concatenation, comparison, access, iteration, substrings, replacement, and string/number conversions.
- [Chapter 4: Expressions](ch04.pdf) — assignment, arithmetic, comparison, logical, increment/decrement, compound assignment, bitwise, ternary, and operator precedence.
- [Chapter 5: Control Flow](ch05.pdf) — `if`, `while`, `do-while`, `break`, `continue`, `for` loops, and `switch` statements.
- [Chapter 6: Functions](ch06.pdf) — declarations vs. definitions, parameters, pass-by-value and pass-by-reference, `const`, default parameters, overloading, recursion, function pointers, and operator functions.
- [Chapter 7: Containers](ch07.pdf) — `std::array`, `std::vector`, and iterating through containers with loops and iterators.
- [Chapter 8: I/O Streams](ch08.pdf) — stream manipulators, string streams, file streams, and putting it all together.
- [Chapter 9: std::format and std::print](ch09.pdf) — modern C++23 formatted output with `std::format`, `std::print`, and `std::println`.
- [Chapter 10: Exceptions](ch10.pdf) — throwing and catching exceptions, stack unwinding, `noexcept`, and `std::expected`.
- [Chapter 11: Classes](ch11.pdf) — from structs to classes, access specifiers, constructors, destructors, member functions, the `this` pointer, and operator overloading.
- [Chapter 12: Memory Management](ch12.pdf) — stack vs. heap, pointers, `new` and `delete`, memory leaks, dangling pointers, smart pointers, and move semantics.
- [Chapter 13: Special Members and Friends](ch13.pdf) — the rule of five, defaulted and deleted functions, the rule of zero, and friend functions/classes.
- [Chapter 14: Odds and Ends](ch14.pdf) — `exit()`, `extern "C"`, numbers and casting, time, and random numbers.
